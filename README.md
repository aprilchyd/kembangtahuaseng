âš™ï¸ Penggunaan & Kustomisasi
Website ini sangat mudah dikustomisasi sesuai kebutuhan bisnis.

Mengubah Foto Menu
Saat ini menggunakan gambar placeholder. Untuk menggunakan foto asli produk:

Siapkan foto menu Anda (format .jpg atau .png).
Masukkan foto ke dalam folder assets/images/ (atau folder yang Anda buat).
Cari tag <img> di dalam kode HTML.
<!-- Contoh --><img src="https://picsum.photos/seed/tahujahe/400/300" ...>
Ganti link tersebut dengan path foto lokal Anda.
<!-- Hasil --><img src="assets/images/tahu-jahe-asli.jpg" ...>
Mengubah Informasi Kontak
Cari bagian komentar <!-- CTA Section --> di index.html dan ubah link href pada tombol WhatsApp dan Maps.

Mengubah Harga & Nama Menu
Edit langsung teks di dalam <div class="menu-info"> sesuai dengan menu terbaru.
